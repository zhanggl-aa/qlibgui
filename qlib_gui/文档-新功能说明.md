# QLib GUI 量化研究平台 - 新增功能说明

## 概述

QLib GUI 平台已扩展为一个完整的量化研究解决方案，新增了两个核心功能页面：

1. **因子分析页面** - 分析机器学习模型的因子重要性和暴露程度
2. **投资组合优化页面** - 基于现代投资组合理论进行资产配置优化

## 功能介绍

### 因子分析页面

#### 功能特点：
- **模型选择**：支持选择已训练的模型进行因子分析
- **因子重要性分析**：
  - 可视化展示因子重要性排序图表
  - 详细的因子重要性数据表
  - 累加贡献度计算

- **因子暴露分析**：
  - 因子暴露时间序列图表
  - 因子暴露热图（模拟）
  - 支持市场、市值、动量、价值、盈利等因子

- **因子相关性分析**：
  - 因子相关性矩阵
  - 详细的相关系数表

- **风险因子分析**：
  - 风险分解图表
  - 风险统计数据（波动率、VaR、最大回撤）

- **报告导出**：支持将分析结果导出为 Excel 文件

#### 使用流程：
1. 从模型选择下拉框选择已训练的模型
2. 点击"分析因子"按钮
3. 查看各项因子分析结果
4. 点击"导出报告"按钮保存分析结果

---

### 投资组合优化页面

#### 功能特点：
- **模型选择**：支持选择已训练的模型进行组合优化
- **优化目标**：
  - 最小方差组合
  - 最大夏普比率
  - 最大化收益
  - 效用函数
  - 风险平价

- **约束条件**：
  - 单资产最大权重
  - 单资产最小权重
  - 目标收益率
  - 最大波动率约束

- **股票池配置**：
  - 市场选择（CSI300、CSI500、CSI800、全市场）
  - 股票数量限制

- **可视化展示**：
  - **组合权重**：权重饼图和详细数据表
  - **有效前沿**：风险收益权衡图表
  - **回测表现**：组合与基准的回测对比
  - **风险分析**：风险因子分解

- **优化结果导出**：支持将最优组合导出为 Excel 文件

#### 使用流程：
1. 选择模型和优化目标
2. 配置约束条件和股票池
3. 点击"优化组合"按钮
4. 查看优化结果和可视化图表
5. 点击"导出组合"按钮保存投资组合

## 技术实现

### 因子分析方法

使用模拟数据实现了因子分析的主要功能。实际应用中，应集成 QLib 的以下功能：

1. **特征重要性计算**：
   - 对于树模型，使用模型内置的 feature_importances_ 属性
   - 对于神经网络，可以使用 SHAP 值或特征消融分析

2. **因子暴露计算**：
   - 使用 QLib 的 handler 接口获取原始因子数据
   - 计算模型预测与因子值的相关性

3. **因子相关性分析**：
   - 使用 pandas 的 corr() 方法计算相关系数矩阵
   - 可视化使用热力图

4. **风险分解**：
   - 计算因子暴露的波动率
   - 使用 VaR（Value at Risk）方法计算风险值

### 投资组合优化方法

投资组合优化使用模拟数据实现。实际应用中，可以使用：

1. **经典马科维茨模型**：
   - 协方差矩阵计算
   - 均值-方差优化

2. **现代优化技术**：
   - 风险平价模型
   - 层次风险平价
   - 机器学习驱动的优化

3. **约束处理**：
   - 线性不等式约束
   - 整数约束（如需）

4. **回测和评估**：
   - 使用 QLib 的回测引擎
   - 计算组合业绩指标
